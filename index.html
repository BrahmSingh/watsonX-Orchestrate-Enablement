About the Bootcamp

This is a two day bootcamp. During this bootcamp, you will understand how to identify a customer's need for the Watson Orchestrate solution. During this boot camp, you will be assigned to a group and as a group, you will take a business case, discuss and white board the solution and then present it to the audience in the bootcamp. You will use the knowledge of Hands-on-Labs to build the solution.

Overview of IBM watsonx Orchestrate

IBM wastonx Orchestrate (WxO) offers skill sets that you can use to handle your mundane routine tasks to get your work faster. WxO uses skills to complete tasks. These skills can accomplish one task or a collection of actions sequenced. It can also can process your quesries using natural language processing.

Skill Sets

Skill sets are a collection of skills which can help you automate the tasks. WxO makes these skills sets more intelligent by adding the cognitivie inferencing. There are two categoies of skills sets: personal skill sets and team skill sets. Personal skills sets can be added by you. Team skill sets are added by an admin. Using these skill sets, you can then create mundane tasks like create a job requisition, filtering the applicants based on the required factors for the job and so on. You can search the skill sets from the catalog or just by querying WxO for a type of skill set.
